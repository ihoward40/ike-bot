# IKE-BOT Documentation

This directory contains comprehensive documentation for the IKE-BOT API.

## Available Documentation

### [API.md](./API.md)
Complete API reference documentation including:
- Authentication endpoints
- CRUD operations for filings, documents, and logs
- Notion integration endpoints
- Webhook endpoints
- Request/response examples
- Error handling

### [SETUP.md](./SETUP.md)
Detailed setup and installation guide covering:
- Prerequisites and requirements
- Local installation steps
- Supabase database setup
- Notion integration configuration
- Make.com webhook setup
- Development and production commands
- Troubleshooting common issues

### [DEPLOYMENT.md](./DEPLOYMENT.md)
Production deployment instructions for:
- Docker deployment
- Cloud platforms (Heroku, DigitalOcean, AWS)
- Environment variable configuration
- CI/CD with GitHub Actions
- Security checklist
- Monitoring and logging

### [API_COLLECTION.json](./API_COLLECTION.json)
Postman/Thunder Client collection for testing the API. Import this file into your API client to quickly test all endpoints.

## Quick Links
